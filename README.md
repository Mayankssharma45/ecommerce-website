# 🛒 E-Commerce Website

A basic E-commerce website built using PHP with product catalogue and order management features.

---

## 🚀 Features

- 🛍️ Product Catalogue (Display products with images)
- ➕ Add to Cart functionality
- 🛒 Shopping Cart system
- 📦 Order management
- 💻 Simple and clean UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

--
